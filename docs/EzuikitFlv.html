EzuikitFlv

EzuikitFlv.js 播放flv直播流， 支持播放h264/h265的直播视频流

Constructor

new EzuikitFlv(options)

Example
```js
const flv = new EzuikitFlv({
 id: "EleId",
 url: "https://open.ys7.com/v3/openlive/E71992743_1_1.flv"
})
```
